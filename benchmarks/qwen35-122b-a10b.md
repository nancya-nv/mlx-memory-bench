# Qwen3.5-122B-A10B (MoE, 10B active)

Large MoE model, 10B active parameters. 65GB at 4-bit -- only fits on 128GB+ machines.

## KV cache sweep

```bash
python3 mlx_memory_bench.py kv-sweep \
    --model mlx-community/Qwen3.5-122B-A10B-4bit \
    --ctx 2048,4096,8192,16384,32768,65536,131072 \
    --output results/kv_sweep_qwen35_122b.json
```

## Notes
- 65GB model weights -- leaves ~63GB for KV cache + OS on 128GB machine
- Tests the upper bound of what M5 Max can handle
- KV cache at 128K context uses ~20GB, total ~86GB peak
- OOM at ~135K+ context
